候補

使い方
  1. 検索欄にタイトルの一部を入力 → 候補が表示されます。
  2. 候補の「追加」を押すと、左のお気に入りに保存されます（ブラウザのローカルストレージ）。
  3. 保存したタイトルはクリックで cal.syoboi.jp/tid/<TID> のページを開きます。
  4. 書き出し/読み込みでバックアップ・移行も可能です（JSON）。

※ 検索データは しょぼいカレンダーのJSON API (TitleSearch) をJSONPで利用しています。
